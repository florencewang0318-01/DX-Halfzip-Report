Development Translation
One shared light-outdoor base, then split into Female and Male extension paths
Both genders should still start from the same light-outdoor scene logic, but the next layer should separate into a more active female expression and a more utility-led male expression.
Shared Base Female Extension Male Extension
01 / Core Base
先把凉感、透气、轻量做成品类基础
这是 27SS 防晒衣最该先做稳的一层功能与面料底盘，不属于某个单独性别。
判断

轻薄平纹、微皱尼龙、空气感梭织、轻量防晒壳感面料，优先解决夏季轻户外的低负担穿着。
作用
作为男女共用的开发底盘，先把舒适、防晒与轻体感做扎实，再往各自方向延展。
